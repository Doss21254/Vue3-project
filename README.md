
# Vue3 用户数据表格CRUD实战项目
## 技术栈
Vue3 + Vite + TypeScript + <script setup> 组合式API
## 项目功能
1. 简易待办清单：ref响应式、v-model双向绑定、事件修饰符
2. 用户管理表格完整CRUD：新增/编辑复用弹窗、前端搜索过滤、列表渲染
## 本地启动教程
1. 克隆仓库
git clone https://github.com/Doss21254/Vue3-project.git
2. 进入项目文件夹
cd Vue3-project
3. 安装依赖包
npm install
4. 启动本地开发服务
npm run dev
